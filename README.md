# See My Password

A simple Kotlin app with basic password reset helper functions.

This repository contains example functions for requesting a password reset
and changing a password. These are minimal examples and are NOT secure for
production use. Improvements needed before production:
- Proper email sending (SMTP, APIs)
- Secure OTP/reset link generation
- Password hashing and secure storage
- Input validation and rate limiting

- src/main/kotlin/PasswordReset.kt — basic functions
